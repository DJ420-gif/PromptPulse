01

What Claude Memory Is

Anthropic rolled out persistent memory to all Claude users in April 2026, making it free at every tier including the free tier. The timing was strategic — Anthropic made memory import free on March 2, precisely during the QuitGPT surge, reducing friction for users switching from ChatGPT. Claude memory uses a system prompt approach: remembered facts are injected at the start of each conversation as structured context. You can view, edit and delete every individual memory from the settings panel. The transparency is intentional — Anthropic's constitutional AI philosophy applies to memory the same way it applies to responses. You should be able to see and control what Claude knows about you, not just use an opaque black box.

02

What Claude Actually Remembers

Claude memory stores facts rather than episodic conversation transcripts. What it stores well: your name, professional role, skills, communication style preferences, ongoing projects, recurring topics you care about, and explicit instructions about how you want Claude to behave. What it does not store: full conversation transcripts, the specific wording of past exchanges, or individual message content. Claude with memory is a better personal assistant — it knows you are a frontend developer who prefers TypeScript and dislikes verbose explanations. It is not a conversational archive. For reference across conversations, Claude Projects stores documents and instructions persistently at the project level — a different feature from memory that serves a different need.

03

Claude Memory vs ChatGPT vs Gemini

All three major AI assistants have persistent memory as of April 2026, with different implementations. ChatGPT memory: stores facts and preferences with user-visible management, similar to Claude with less granular per-memory editing on mobile. Claude memory: structured fact storage, full edit and delete per item, free at all tiers, system-prompt injection mechanism visible in the settings. Gemini memory: integrates with Google account context including Calendar, Gmail and Drive — broader context at the cost of Google data practice acceptance. For pure AI assistant memory control, Claude and ChatGPT are comparable. Claude's advantage is the transparency and granularity model. Gemini's advantage is Google ecosystem depth.

04

Memory Import From ChatGPT — The Switching Feature

The memory import feature made free on March 2 2026 lets you export your ChatGPT memory file and import it into Claude. Export from ChatGPT Settings, extract the memory file, upload to Claude's memory import tool. Core facts — name, role, projects, preferences — typically transfer cleanly. Some context requires manual re-entry due to schema differences. Not every memory transfers perfectly, but the primary switching barrier — losing months of established AI context — is eliminated. This was Anthropic's explicit strategic intent during the QuitGPT week: make switching cost zero at exactly the moment motivation to switch was highest.

Frequently Asked Questions

Is Claude memory free?
Yes — free at all tiers including the free tier from March 2 2026. View, edit and delete individual memories from Claude Settings on web and mobile.
What does Claude remember across conversations?
Your name, role, skills, preferences, ongoing projects and explicit instructions. Does not store full conversation transcripts or individual message wording. Fact-based memory, not conversation archive.
Claude memory vs ChatGPT memory?
Both store facts with user-visible management. Claude offers more granular per-memory editing. Gemini adds Google account context. Choose based on your primary platform and data preference.
How do I import ChatGPT memory to Claude?
Export data from ChatGPT Settings, extract the memory file, upload via Claude memory import tool. Core facts transfer cleanly. Schema differences may require manual re-entry for some context.
Can I delete Claude memory?
Yes — view, edit and delete individual memories from Claude Settings. You can also pause memory collection without deleting existing memories. Full transparency and control.
