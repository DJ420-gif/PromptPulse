How the Leak Happened and What Was Found

On March 31 2026, an npm packaging error in Claude Code exposed a source map containing 512,000 lines of TypeScript. Anthropic filed a DMCA takedown on GitHub, but researchers had already mirrored and analysed the code. Inside the codebase: references to an unreleased product called Conway. UI components, a trigger engine, a CNW extension format, and persistent memory systems — none of which exist in any current Anthropic product. The name Conway appears throughout in a way consistent with active development, not experimental prototyping. Anthropic has not commented on Conway specifically but confirmed the DMCA request.

02

Conway's Six Core Components

The leaked code reveals six identifiable Conway components. Trigger engine: watches calendar entries, email arrivals, file changes and schedules — acts autonomously without a user prompt. Persistent memory: maintains a model of the user's workflows and decision patterns across sessions. CNW extension format: .cnw.zip plugin files that define custom tools, UI tabs and context handlers — proprietary, Conway-specific, creating platform lock-in. Permission policies: governance layers on Anthropic's infrastructure. Background execution: runs continuously without UI interaction. Conway-Claude Code integration: tight coupling with Claude Code's existing tool and agent infrastructure shared with Claude Managed Agents.

03

The Open-Then-Proprietary Platform Strategy

The leaked code reveals a two-layer platform strategy. Layer one is MCP — genuinely open, adopted by all major AI providers, portable across frameworks. Anthropic uses MCP at the base tool connectivity layer. Layer two is CNW extensions — proprietary, Conway-specific, locked to Anthropic's platform. This is the classic platform playbook: attract developers with an open standard at the base (MCP), then add a proprietary layer on top (CNW) where commercial value accumulates. Developers who invest in CNW extensions build tooling that only works inside Conway — the iOS app store dynamic applied to AI agent plugins. The connection to Claude Managed Agents (April 8) and Claude Code suggests Conway is the consumer-facing product built on the same shared infrastructure.
