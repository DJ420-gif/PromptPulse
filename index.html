The Framework

The 6-Part Prompt
Formula

Every world-class AI output starts with these 6 layers. Miss one and your results suffer. Use all six and you'll rarely need to re-prompt.

🎭
Layer 1
Role
Tell AI exactly who it is. Seniority, expertise, mindset. This anchors the entire conversation.
"You are a senior fullstack engineer with 10+ years building production SaaS..."
🏢
Layer 2
Context
Project name, description, target users, business model, expected scale. Business context = better architecture.
"B2B SaaS for restaurant owners. Non-technical users aged 35–55. 10K users."
🛠
Layer 3
Tech Stack
Exact framework versions prevent deprecated APIs and incompatible patterns from ever appearing.
"Next.js 15, TypeScript 5.4, Tailwind 3.4, Prisma 5, Supabase, Vercel"
🚫
Layer 4
Constraints
The most underrated layer. Tell AI what it must never do. This eliminates entire categories of bad output.
"NEVER: any, var, console.log, partial code, magic numbers, inline styles"
📦
Layer 5
Output Format
Define exactly how you want the response structured — files, comments, explanations, next steps.
"Full files only. JSDoc comments. After code: WHAT / WHY / EDGE CASES / NEXT"
🎯
Layer 6
The Task
Be surgical. Component name, data shape, user interactions, what it connects to, what success looks like.
"Build a MRR chart component using Recharts + React Query v5 with loading skeleton..."
